# 設計理念：末世科技監控系統

## 選定設計方向：**賽博朋克末世感**

### 設計哲學
融合高科技賽博朋克美學與末世警示主題，打造沉浸式數據監控體驗。設計強調科技感、緊張感和視覺層次。

---

## 核心設計原則

### 1. **視覺層次與信息架構**
- 中央雷達為核心焦點（高科技感）
- 左側新聞流為背景信息（持續更新）
- 右側統計數據為快速參考
- 下方猝死明細為詳細數據
- 超連結區塊為補充資訊

### 2. **色彩系統**
- **主色**：青色 `#00f3ff` - 科技感、警示
- **警告色**：紅色 `#ff003c` - 危險、緊急
- **次色**：金色 `#f5d300` - 末世感、轉折
- **背景**：深黑 `#030305` - 深空感、對比
- **文字**：淺灰 `#eeeeee` - 易讀性

### 3. **排版系統**
- **標題字體**：Orbitron（未來感）
- **技術字體**：Share Tech Mono（終端感）
- **正文字體**：Noto Serif TC（可讀性）
- 字重對比：粗體標題 + 細體正文

### 4. **布局範式**
- **桌面版**：3欄布局（左新聞 | 中雷達 | 右統計）
- **手機版**：單欄堆疊（新聞 → 雷達 → 統計 → 明細）
- 響應式斷點：768px
- 流動性設計：無固定寬度限制

### 5. **簽名元素**
1. **掃描線效果**：全屏掃描線營造監視器感
2. **霓虹光暈**：元素邊緣青色/紅色發光
3. **故障文字**：標題偶發性文字故障效果
4. **雷達掃描**：中央旋轉掃描動畫

### 6. **交互哲學**
- **懸停反應**：卡片發光、放大、邊框變亮
- **點擊反饋**：立即視覺反應
- **數據更新**：平滑過渡動畫
- **警告狀態**：紅色閃爍、脈衝效果

### 7. **動畫指南**
- **掃描動畫**：8秒線性旋轉（無限循環）
- **脈衝效果**：2-4秒呼吸式縮放
- **閃爍警告**：1秒快速閃爍
- **故障效果**：3秒隨機抖動
- **過渡時間**：0.2-0.3秒標準過渡

### 8. **排版層級**
```
H1: 3-7rem (Orbitron 900) - 主標題
H2: 1.8rem (Orbitron 700) - 卡片標題
H3: 1.2rem (Orbitron 600) - 小標題
Body: 1rem (Noto Serif TC 400) - 正文
Meta: 0.9rem (Share Tech Mono 400) - 技術信息
```
