## Progressive Awakening System

A gamified spiritual journey website that guides users through levels of understanding:
- **Lv 0 (AWAKENING)**: Introduction on `index.html`
- **Lv 1 (AWARENESS)**: Monitor page unlocked at `monitor.html`
- **Lv 2 (OBSERVER)**: Timeline unlocked at `timeline.html`

---

## Repository Structure

### Main Website (Root Directory)
- `index.html` - Main entry point (Level 0: AWAKENING)
- `monitor.html` - Monitoring page (Level 1: AWARENESS)
- `timeline.html` - Timeline page (Level 2: OBSERVER)
- `Bible/` - Bible-related resources and tools

### Documentation (`docs/`)
Project documentation and management files:
- `project-instructions.md` - Development guidelines and project overview
- `conversation-log.md` - Development history and decisions
- `mission-briefing.html` - Project briefing document

### Data (`data/`)
Raw data files used for statistics and charts:
- `data.txt` - Statistical data

### Scripts (`scripts/`)
Python scripts and automation tools:
- Place Telegram scrapers, data processors, and other backend scripts here
- See `scripts/README.md` for details

### Assets (`assets/`)
Future location for separated CSS, JavaScript, and images:
- `assets/css/` - Stylesheets
- `assets/js/` - JavaScript files
- `assets/images/` - Image files

### Archive (`archive/`)
Unused or sample files for reference:
- Old versions and test files
