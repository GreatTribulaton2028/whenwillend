# Scripts Directory

This directory is for Python scripts and other backend automation tools.

## Purpose

Use this folder to store:
- Telegram scrapers
- Data processing scripts
- Automation tools
- Any other backend scripts needed for the project

## Usage

Scripts in this directory should be self-contained and well-documented.
Add comments explaining what each script does and how to use it.

## Requirements

If your scripts have dependencies, consider adding a `requirements.txt` file to this directory.
